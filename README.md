# Sherwood Legends

Static website for `sherwoodlegends.com`.

## Project

Sherwood Legends is a fantasy MMORPG website for the custom Sherwood Legends realm.

- Creator: Asilzade
- Company: Noblesse Ltc.
- Theme: dark purple, gold, castle fantasy
- Domain: `sherwoodlegends.com`

## Files

- `index.html` - main landing page
- `CNAME` - GitHub Pages custom domain
- `assets/` - site images

## Deploy Target

The intended deployment target is GitHub Pages with the custom domain:

```txt
sherwoodlegends.com
```

Namecheap DNS can be connected after the GitHub Pages repository is published.
